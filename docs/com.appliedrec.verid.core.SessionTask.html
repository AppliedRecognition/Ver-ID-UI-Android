SessionTask

AsyncTask that runs a Ver-ID session

Package com.appliedrec.verid.core

public class SessionTask

Constructor

SessionTask

Introduced in version 1.0.0

public SessionTask(VerID, IImageProviderService, IFaceDetectionService, IResultEvaluationService, IImageWriterService, IVideoEncoderService)

Task constructor

The task periodically retrieves images from the image provider and passes them to the face detection service. The face detection service passes its result to the result evaluation service. Images are written to a file by the image writer service.

Once the result evaluation service deems it has enough information to end the session it completes the task.

As the task runs it sends updates to the session task delegate(s).
